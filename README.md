# Snake Game Using Java

## 📌 Project Overview
Snake Game is a classic arcade game developed using Java. The player controls a snake that moves around the game window, eating food to grow longer. The objective is to achieve the highest score possible without colliding with the walls or the snake's own body.

## 🎮 Features
- Interactive graphical user interface using Java Swing.
- Snake movement controlled by keyboard arrow keys.
- Random food generation.
- Score tracking system.
- Game Over screen when collision occurs.
- Increasing difficulty as the snake grows.

## 🛠️ Technologies Used
- Java
- Java Swing
- AWT (Abstract Window Toolkit)

## 🚀 How to Run
### Prerequisites
- Java Development Kit (JDK 8 or higher)
- Any Java IDE (Eclipse, IntelliJ IDEA, NetBeans) or Command Prompt

### Steps
1. Clone or download the project.
2. Open the project in your preferred Java IDE.
3. Compile and run the `SnakeGame.java` file.

Using Command Prompt:
```bash
javac SnakeGame.java
java SnakeGame
```

## 🎯 Game Controls
| Key | Action |
|------|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |

## 📈 Game Rules
1. Control the snake using the arrow keys.
2. Eat food to increase your score and snake length.
3. Avoid hitting the walls.
4. Avoid colliding with the snake's own body.
5. The game ends when a collision occurs.
